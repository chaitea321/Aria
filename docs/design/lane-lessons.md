# Lane Lessons (self-improvement log)

Append one line whenever a PR is **denied** or sent back for rework, or when a lane worker
hits a recurring snag. These lessons feed back into [`lane-kickoff-prompts.md`](lane-kickoff-prompts.md):
periodically fold the recurring ones into the relevant lane's prompt so the next worker
doesn't repeat them.

Format: `YYYY-MM-DD · <lane> · <PR#/finding> · what went wrong → the rule to add`.
